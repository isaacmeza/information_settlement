# Information and Bargaining through Agents: Experimental Evidence from Mexico's Labor Courts

This paper is the result of a series of RCTs conducted in the Mexico City Labour Court. 


# Repository structure

Our repository is organized in five main sections, each one with a particular directory. Three of them contain all work related to a particular experiment, and the remaining two mix work from different experiments. The main sections are:

- Paper

- ScaleUp

- Reports

- presentations



# RScripts and Do-Files

You will see there are subdirectories for Raw and Clean Data (DB), and for RScripts and DoFiles used to generate content. RScripts are always run from the RScripts directory, while DoFiles are run from the experiment directory by declaring a global variable.  All scripts have the name of the figure or table they are generating.

File cleaning processes are generally done in R and take Raw data as an input and have DB data as an output. However, some slight variable recoding is done in STATA in some cases when running regressions.
