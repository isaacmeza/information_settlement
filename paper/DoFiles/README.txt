This is the (short) documentation of the results for the paper 

# - Information and Bargaining through Agents: Experimental Evidence from Mexico's Labor Courts
# By Joyce Sadka, Enrique Seira, and Christopher Woodruff

The documentation on how the repository is organized can be found on <https://bitbucket.org/IsaacMeza/information_sett/src/master> (private repository)

Data has been pre-processed and cleaned with R. However further processing, and generation of relevant variables is also done in STATA.

1) To change the directory, the relevant path has to be modified in 
	global_directory_paper.do

2) The further cleaning of data is run through 
	master_cleaning.do

3) The main results are run through
	master_main.do

4) Results from the appendix are run through
	master_appendix.do
